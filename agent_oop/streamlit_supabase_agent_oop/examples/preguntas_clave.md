# Preguntas clave para el Streamlit Agent

## Inventario

- ¿Cuántas unidades disponibles hay por proyecto?
- ¿Cuál es el stock por tipología?
- ¿Qué proyectos tienen más unidades por vender?
- ¿Cuál es el precio promedio por m2 por proyecto?
- ¿Cuál es el precio promedio por m2 por distrito?

## Comercial

- ¿Cuántas separaciones hubo este mes?
- ¿Qué asesor tiene más ventas?
- ¿Qué medio de captación trae más ventas?
- ¿Cuál es el ranking de asesores?
- ¿Qué proyecto convierte mejor?

## Finanzas / Cobranzas

- ¿Cuánto se ha cobrado por proyecto?
- ¿Cuánto falta cobrar?
- ¿Qué proyectos tienen mayor brecha de cobranza?
- ¿Cuál es el porcentaje cobrado por proyecto?

## Calidad de datos

- ¿Qué ventas no tienen documento de cliente?
- ¿Qué clientes están duplicados?
- ¿Qué contratos tienen diferencias con el sistema?
- ¿Qué proformas no cruzan con pagos?

## Geo / Cliente

- ¿Qué distritos concentran más compradores?
- ¿Cuáles son los proyectos por distrito?
- ¿Qué zonas tienen mayor precio por m2?
